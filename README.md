# KeyMouse Heatmap

KeyMouse Heatmap 是一个 Windows 键盘/鼠标使用统计工具：键盘热力图、鼠标热力图、鼠标移动距离、输入速度、峰值时段、排行、分组、应用详情（按应用汇总按键与使用时长），并支持悬浮窗实时显示。

当前版本：v1.8.1

## 主要功能

- 键盘热力图：真实 104 键布局，支持左右 Shift/Ctrl/Alt/Win 分开统计与实时高亮。
- 鼠标热力图：左键、右键、中键、侧键、滚轮上/下统计与高亮。
- 鼠标移动距离：按屏幕像素估算并显示为 m / km。
- 总览页：键盘/鼠标热力图 + Top10（排行/分组/应用）。
- 排行页：按键次数排行，带占比统计条。
- 分组页：字母区、数字区、功能键区、方向键区、小数字键盘、修饰键、鼠标等分组统计，带统计条。
- 应用详情：一应用一行，显示应用图标、使用时长、总次数、按键种类、占比统计条、常用按键摘要。
- 速度页：当前分钟、当前小时、平均速度。
- 峰值页：按小时查看键盘/鼠标峰值。
- 悬浮窗：显示最近按键、正在按住、速度（可开关）。
- 导出：CSV、PNG、HTML 报告。
- 主题：浅色 / 深色 / 赛博蓝 / 极简。

## 数据与配置位置

从 v1.5.0 起，数据与配置固定保存到：

```text
%LocalAppData%\KeyMouseHeatmap\data
```

更新不会清空原来的统计数据与配置，不同版本可共用同一份数据。

首次启动新版本时，程序会自动尝试把旧版本 exe 同目录下的 `data/*.json` 迁移到上面的固定目录（复制迁移，不删除旧文件）。

## 下载与运行（推荐）

Release 通常提供两种包：

- 免安装 .NET（推荐多数用户）：`portable-compressed-win-x64`
  直接运行压缩包内的 `KeyMouseHeatmap.exe`
- 体积更小（需要 .NET 9 Desktop Runtime）：`lite-win-x64`

## 重要说明（性能与实时性）

- 本软件使用 Raw Input 统计键盘与鼠标按钮/滚轮，避免全局低级鼠标钩子导致的光标卡顿。
- 为保证“快速按键/多键同时按下”也能保持流畅，界面高亮与悬浮窗采用合并刷新（约 60fps 上限），避免每个按键都强制同步重绘造成输入延迟。

## 打包（开发者）

需要 .NET 9 SDK：

```powershell
dotnet restore
dotnet build
```

发布（示例）：

```powershell
dotnet publish -c Release -r win-x64 --self-contained false -p:PublishSingleFile=false -o .\publish\lite-win-x64
dotnet publish -c Release -r win-x64 --self-contained true  -p:PublishSingleFile=true  -p:IncludeNativeLibrariesForSelfExtract=true -o .\publish\portable-compressed-win-x64
```

也可以直接运行仓库内的发布脚本（如果有）。

## 更新记录（节选）


```text
publish\lite-win-x64
```

## 隐私说明

本工具只统计：

- 按键名称
- 鼠标按键名称
- 次数
- 时间分布
- 应用进程名、可执行文件路径和运行/使用时长（用于应用详情图标、使用时长和应用统计）

本工具不记录：

- 具体输入文本内容
- 密码
- 聊天内容
- 网页内容
- 剪贴板内容

所有数据默认保存在本机 `%LocalAppData%\KeyMouseHeatmap\data`。

## 注意

本工具使用 Windows Raw Input 接收键盘和鼠标事件，不拦截、不改写、不阻止其他应用的输入和快捷键。部分安全软件仍可能对输入统计工具提示风险，这是同类工具常见情况。

请不要将本工具用于未经允许的监控、记录或侵犯他人隐私的场景。
